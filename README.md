# Flappy-Bird-Clone
A simple clone of the classic Flappy Bird game built using Python and Pygame. Designed as a beginner-level project to understand game loops, event handling, and collision detection.

## Features
- Endless side-scrolling gameplay
- Scoring system
- Game over screen
- Simple graphics using Pygame

## Installation

1. Clone the repository :
git clone https://github.com/yourusername/flappy-bird-python.git

2. Navigate to the project directory :
cd flappy-bird-python

3. Install dependencies :
pip install pygame

4. Run the game :
python flappy_bird.py

####  **Usage / How to Play**
Instructions for using the project.


## How to Play

- Press the spacebar to flap the bird.
- Avoid hitting the pipes.
- Try to beat your high score!
